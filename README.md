# CraigslistApartment_Vancouver_Skytrain
An app to search rental apartments in Vancouver and determine the commute time on Skytrain.  
Currently the commute destination address is hard-coded as its just for me.
I had to add random headers and randomized delays in the craigslist search to overcome being flagged as a bot.
The delays make searching slower than ideal, but its still waaaay easier than manually figuring out if a given listing 
is within an acceptable commute time.
